**Heuristic:** MC
**Concept (one sentence):** Metacognition is a closed control cycle — set goal, attempt, monitor, adjust, evaluate — and the learner must run every segment, or an external regulator keeps the loop running while the learner's own capacity atrophies.
**Figure type:** cycle (closed control loop)

At the center of the ring, place a small node representing the learner as the agent who must close the loop. Draw a single-headed arrow from the central learner node out to each ring node to indicate the learner powers every segment. Keep the ring clean and circular so the return-to-start is unmistakable. Color the five ring nodes blue as the structural cycle, the central learner-agent node bluish green to mark it as the irreducible owner, and the connecting arrows black.
