**Heuristic:** VG
**Concept (one sentence):** Instructional clarity splits into an AI-delegable planned half and a human-only live half, and the same delegation builds the live half for a novice through struggle but is mere time-saving for an expert.

## Figure 2 — The live-half feedback loop  ·  Important
**Heuristic:** MC
**Concept (one sentence):** Expert clarity is a real-time loop — explain, check, detect, adjust, re-explain — whose detection step is the heart and which a script-bound teacher structurally cannot run.
**Figure type:** cycle (real-time feedback loop with emphasized detection node)

**Block 1 — Illustrae paste block:** Create a single-column cyclical diagram, 89mm wide at 300 DPI, vector, white background, with five nodes in a ring connected by single-headed arrows flowing one direction to close the loop: explain, check, detect, adjust, re-explain, returning to explain. Emphasize the detect node as the heart of the loop — render it larger and with a heavier border than the others. Draw a separate script block positioned outside the ring with a single-headed arrow that reaches only the explain node and stops there, with a perpendicular stop bar preventing it from connecting to the detect node — signifying the script supplies the explanation but cannot perform detection. Color the loop nodes blue, the emphasized detect node bluish green, and the external script block with its blocked connector vermillion. Keep the ring clean and circular so the cycle reads clearly. Flat nodes, uniform thin strokes except the heavier detect-node border. Request an unannotated vector diagram with no baked-in text.

- **O:** Closed circular ring of five nodes, single-headed arrows; detect node enlarged with heavier border; external script block with a single-headed arrow to explain and a ⊣ stop bar at the detect boundary.
